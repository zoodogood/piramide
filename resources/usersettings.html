Подробности Если вы серьёзно занялись оптимизацией вам может быть интересна статистика.
С момента активации этой функции будет сохранятся среднее количество шагов за которое код может одержать победу

Как это работает?
1. Когда вы одержали победу, незаметно для вас алгоритм ещё раз выполниться в тестовом полигоне со статичными условиями, которые были подобраны мной для наибольшего баланса.
2. Результат этого будет сохранён в браузере и в будущем может использоваться для сравнения
