ПОЯСНЕНИЕ ЗАДАЧИ

Напишите алгоритм, чтобы собрать все плитки внутри одного массива, башни.
Напрямую взаимодействовать с ними никак нельзя. Используйте метод класса game — step()
Функция "шаг" принимает два аргумента: номер массива с которого перенести плитку и того, в который её перекинуть.
Хотя прямая работа с башнями невозможна — получить их список можно с помощью свойства game.list

Прочитайте

1. Используйте консоль разработчика (по умолчанию F12) Это поможет получить информацию о процессе выполнения вашего алгоритма.
2. Воспользуйтесь библиотекой.
3. Посетите g страницу Гитхаба.
4. Помните, Ваш код и статистика сохраняются даже при перезагрузке страницы.
5. Осторожнее с бесконечными циклами, их условие всегда верно. Зависание страницы — явный признак такого цикла.
